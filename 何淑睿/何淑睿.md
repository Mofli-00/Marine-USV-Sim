# 4.3

### 1. 基础场景搭建

在 Unity 2022.3.62f2c1（3D Core 项目）中从零开始搭建了海洋竞速游戏的基础场景，主要完成以下内容：

- 创建大尺寸 Plane 作为海面
- 创建 Cube 作为船体占位模型，调整尺寸比例使其具备船形外观
- 为船体挂载 Rigidbody 组件，启用物理仿真，并冻结 X/Z 轴旋转防止船体翻倒

### 2. 场景资产导入与地形丰富

**Asset Store 资产导入**
- 通过 Unity Package Manager 导入 Free Island Collection
- 将岛屿 Terrain 叠加至现有场景，调整 Y 轴位置使岛屿自然从海面露出
- 发现 Asset Store 在国内网络环境下访问困难，加载速度极慢，后续将不再依赖该方式获取资产
